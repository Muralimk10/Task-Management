🔹 About TaskFlow
TaskFlow is designed to help individuals and teams manage workflows efficiently. With an intuitive Kanban-style interface, users can easily visualize progress, prioritize tasks, and stay organized.

🎯 Our Mission
To simplify task management and improve productivity through a clean, user-friendly experience. Whether you're working solo or collaborating with a team, TaskFlow helps you stay on track and achieve your goals.

⚡ Key Features:
• Fast & responsive interface with smooth drag-and-drop functionality
• Interactive and user-friendly design
• Team-friendly workflow concept
• Goal-oriented task organization

💡 What I Learned:
• Building interactive UI using JavaScript
• Creating responsive layouts with CSS
• Improving UX and performance
• Writing clean and maintainable code
